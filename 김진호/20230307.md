삼성헬스 - 스트레스

```sql
현재 삼성헬스에서 스트레스 데이터가 확인되면 1분단위 데이터를 저장함. 초단위 데이터는 
별도로 저장되지않고, 1분간 평균,최대값, 최소값이 json에 배열값으로 저장. 
해당 평균값으로 0~9, 10~19, 20~29, 30~39, 40~49, 50~59분 데이터를 모아서 평균을 낸 
값을 막대그래프로 보여주고 있음. 또한 위쪽에 보이는 값또한 10분당 데이터를 다시 평균낸 값.
```

따라서 차트에 사용하는 방식은
하단 데이터 = 10분별 수치
상단 데이터 = 일별 평균
으로 차트를 구성.

해당 데이터를 얻으려면(파이썬기준)

1. zip파일에서 com.samsung.shealth.stress.*.csv csv 라이브러리로 값 받아오기
2. csv 데이터에 있는 start_time, binning_data, score 데이터 추출
3. score와 start_time 데이터로 일별 평균 데이터 구해서 DB에 값 저장
4_1. binning_data이름으로 ./jsons/com.samsung.shealth.stress 에 있는 json 찾기
4_2. 해당 json에서 start_time을 기준으로 알맞은 10분간 데이터에 합산하여 DB에 값 저장

필요한 기술(+라이브러리)
csv 라이브러리 : csv 파일을 행단위로 배열로 받아오는 라이브러리
